# 💻 Residuos Informáticos

Los residuos electrónicos son dispositivos que ya no se usan y pueden contener materiales contaminantes.

![Residuos electrónicos](../imagenes/residuos.jpg)

## Impacto ambiental

| Dispositivo       | Problema ambiental          |
|-------------------|----------------------------|
| Ordenadores       | Metales pesados            |
| Teléfonos móviles | Baterías contaminantes     |
| Monitores         | Componentes tóxicos        |

> El reciclaje responsable es fundamental para minimizar los daños.
