# 🌍 Informática Ambiental

Proyecto sobre la relación entre la informática y el medio ambiente.

Este trabajo analiza cómo la tecnología afecta al medio ambiente y qué soluciones existen para reducir su impacto.

---

## 🎯 Objetivo del proyecto

El objetivo de este proyecto es estudiar el impacto de la tecnología en el medio ambiente y conocer soluciones que permitan un uso más sostenible de los dispositivos informáticos.
